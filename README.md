# Kunwar Dairy

Customer-facing website and dairy subscription platform for **Kunwar Dairy** — farm fresh milk delivery, powered by the Shree Shyam Dairy Farm operations team.

**Live:** [kunwardairy.com](https://kunwardairy.com)

## Features

- Responsive Design
- Modern UI/UX
- Hero Banner
- About Section
- Product Showcase
- Why Choose Us Section
- Call To Action
- Contact Information
- SEO Friendly Structure
- Fast Performance

## Tech Stack

- Next.js 15
- React 19
- Tailwind CSS
- JavaScript
- Turbopack

## Project Structure

```text
app/
components/
lib/
public/
├── icons/
├── images/
└── logos/
```

## Getting Started

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

Open:

```text
http://localhost:3000
```

### Production Build

```bash
npm run build
npm start
```
