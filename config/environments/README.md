# Environment configuration

Copy `.env.example` to the appropriate file:

| File              | Environment  |
| ----------------- | ------------ |
| `.env.local`      | development  |
| `.env.test`       | testing (CI) |
| `.env.staging`    | staging      |
| `.env.production` | production   |

Set `APP_ENV` to match: `development` | `testing` | `staging` | `production`.
