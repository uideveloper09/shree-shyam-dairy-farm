# Release Notes — v0.1.0

**Date:** June 2025  
**Codename:** Enterprise Platform

Initial release of the unified Shree Shyam Dairy Farm ERP with enterprise-grade platforms.

## Platforms

### E-commerce

- Storefront with cart, Razorpay payments, milk subscriptions
- Customer account, addresses, order history

### Farm Intelligence

- IoT sensor ingest, MQTT bridge, edge gateway
- Autonomy engine, weather, CCTV, AI vision
- Farm AI chat, voice, agent, predictions
- Admin dashboard at `/admin/farm`

### Enterprise Security

- JWT auth with opaque refresh tokens
- RBAC/ABAC, 2FA, OTP, Google OAuth, WebAuthn
- Audit logs, GDPR export/delete
- Security dashboard at `/admin/security`

### Mobile PWA

- Role apps: customer, delivery, farm, vet, owner
- Offline sync, push, GPS, camera, barcode, biometric

### Public API & Developer Platform

- REST API v1 with API keys and scopes
- OpenAPI spec, Swagger UI, TypeScript SDK
- Webhooks with HMAC signatures and retry worker

### Multi-Tenant SaaS

- Branding, themes, custom domains
- Stripe + Razorpay tenant billing
- Usage metering and analytics
- Tenant admin at `/admin/tenant`

### DevOps

- Docker + Nginx production stack
- Kubernetes manifests with HPA
- GitHub Actions CI/CD
- Health checks, Prometheus metrics, backups

## Stack

- Next.js 16.2, React 19, Tailwind v4
- Prisma 6 + PostgreSQL (Neon)
- Redis, BullMQ, MQTT, OpenAI

## Known limitations

- `@sentry/nextjs` pending Next.js 16 peer support — structured logging workaround
- MQTT and BullMQ workers require external hosting (not Vercel serverless)
- Auth.js migration not started — custom JWT in production
- WebAuthn signature verification is challenge-based MVP

## Upgrade notes

```bash
npm ci
npx prisma db push
npm run db:seed-tenant
```
